# GAT Release Notes

At Quantify Labs, we're constantly working to improve GAT and provide you with the best possible experience. Here you'll find a chronological list of updates, new features, and bug fixes.

## Latest Release

### Version 2.5.0 (October 15, 2024)

<Note>
This release introduces advanced predictive analytics and enhances our natural language processing capabilities.
</Note>

#### New Features

<CodeGroup>
  <CodeBlock title="Predictive Analytics">
    ```
    - Time series forecasting for key business metrics
    - Anomaly detection in data patterns
    - "What-if" scenario modeling
    ```
  </CodeBlock>
  <CodeBlock title="Enhanced NLP">
    ```
    - Support for complex, multi-part questions
    - Improved context understanding in follow-up queries
    - Expanded industry-specific terminology recognition
    ```
  </CodeBlock>
</CodeGroup>

#### Improvements

- **Performance**: 30% faster query processing times
- **UI/UX**: Redesigned dashboard creation interface for easier customization
- **Data Sources**: Added support for 15 new data source integrations

#### Bug Fixes

- Fixed an issue where certain CSV exports were missing data
- Resolved a rare crash when processing extremely large datasets
- Corrected timezone handling for international users
